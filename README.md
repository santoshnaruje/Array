# DSA Practice - Array

This repository contains solutions to various Array problems in Data Structures and Algorithms (DSA), implemented in C++.

## Structure

The problems are categorized by difficulty:
- **easy/**: Beginner level problems.
- **medium/**: Intermediate level problems.
- **Hard/**: Advanced level problems.

## Build & Run

You can compile individual C++ files using `g++` or use CMake if preferred.

### Using g++
```bash
g++ path/to/source.cpp -o output_binary
./output_binary
```

### Using CMake
This project includes a `CMakeLists.txt` configuration.
```bash
cmake -S . -B cmake-build-debug
cmake --build cmake-build-debug
```
